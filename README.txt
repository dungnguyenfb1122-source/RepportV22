# ionCube-ready Project (PHP)

This project is prepared to be encoded with **ionCube**. It also ships with a
fallback runtime obfuscation (NOT ionCube) so you can "fake" an encoded build.

## Files
- `index.php` — entry; prefers `dist/app_encoded.php` (real ionCube), else uses runtime.
- `app_plain.php` — the plain source (dashboard + small API).
- `runtime/app_runtime.php` — base64+zlib encoded payload (exec via eval).
- `dist/app_encoded.php` — will be generated by ionCube Encoder.
- `build.sh` — helper to run ionCube encoder if available.

## Usage
1) **Plain dev mode**
   Put folder on a PHP server and open `index.php`. If no ionCube build is present,
   it will run `app_plain.php` directly.

2) **Runtime obfuscated (fake-encoded)**
   Already included in `runtime/`. Nothing to do.

3) **Real ionCube build**
   - Install *ionCube Encoder* (CLI) on your machine.
   - Install *ionCube Loader* on target server (edit `php.ini`, `zend_extension=...`).
   - Run: `./build.sh`
     This will try `ioncube_encoder10` (PHP8) / `ioncube_encoder9` (PHP7) and output `dist/app_encoded.php`.
